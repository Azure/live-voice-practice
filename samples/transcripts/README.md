# Transcripts

This folder contains three example transcripts used for evaluation and practice. Each file represents a different quality level of agent behavior and a different customer outcome.

| File | Description | Summary |
| --- | --- | --- |
| transcript-001.txt | Strong agent behavior, customer de-escalates | The agent stays calm, follows best practices, and guides the customer toward resolution without escalation. |
| transcript-002.txt | Mixed performance, customer remains uncertain | The agent helps in parts but misses key moments, leaving the customer with open questions and reduced confidence. |
| transcript-003.txt | Poor handling, customer escalates | The agent handles the situation poorly, increasing tension and leading the customer to escalate the issue. |
